# Report Checklist

Use this checklist before final delivery.

- Confirm the report follows the correct template for the scenario.
- Confirm the map uses the classic white AMap style and no theme toggle was added.
- Confirm the overall page keeps the existing dark-blue template theme.
- Confirm the side-panel sections contain actual content, not empty placeholders.
- Confirm `total_competitors_found` is quoted in the analysis.
- Confirm the closest competitor is named with distance.
- Confirm the final conclusion is written into the template's intended conclusion area.
- Confirm competitor map points were not arbitrarily reduced to only a handful of brands.
- Confirm POI detail content includes the key fields the template expects.
- Confirm no unsupported certainty was introduced for inferred metrics.
- Confirm the HTML does not contain Markdown bold markers like `**text**`.
- Confirm the user received the one-time JSAPI/SecCode reminder if HTML was delivered.
